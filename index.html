How it works

Mind Scout is a progressive web app that collects crucial mental health data and visualizes through the use of an interactive map. This is done through a simple, yet intiuitve, three step process Our three-step process helps lawmakers, public health experts, and the general public to better understand where depression hits

Collection

A user's health data is synced with Mind Scout so that depression linked health factors can be collected. As modern day smart phones automatically collect health information, data such as sleep habits, weight changes, and activity levels can be readily accessed. The data is then stripped of any identifying features, and sent to Mind Scouts website database.

Analysis

The data collected from Mind Scouts users is the run through a machine learning model to determine whether the user has a mental health affliction, such as depression. The current model has trained on a labeld dataset provided by kaggle.com, but there are plans to utilize the data collected by mindscout to implement an unsupervised machine learning model. which would allow mind Scout's predictions to get increasingly accurate as its userbase grows
